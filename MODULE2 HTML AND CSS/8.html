1. Inline CSS

Description: Inline CSS involves applying styles directly to HTML elements using the style attribute. Use Case: Best for applying unique styles to individual elements or for quick styling changes during development or debugging.

Hello, World!

This is a paragraph with inline CSS.

2. Internal CSS

Description: Internal CSS is placed within a tag inside the section of an HTML document. Use Case: Suitable for styling a single document when styles are not reused across multiple pages.

Hello, World!

This is a paragraph with internal CSS.

3. External CSS

Description: External CSS involves linking to a separate .css file that contains all the styles for a website. Use Case: Ideal for larger websites or applications where styles need to be reused across multiple pages.
